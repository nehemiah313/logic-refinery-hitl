# Logic Refinery — HITL Validator

> **"Tinder for Logic" — Human-in-the-Loop Medical Billing Audit Platform**

A full-stack web application for validating AI-generated medical billing logic traces. Human forensic auditors review traces generated by a Neuro-Symbolic pipeline (Phi-4-Mini + GCP ML + Llama 3.1 8B), approve or deny them with a single keystroke, and export Gold Standard JSONL datasets for Bittensor subnet training.

---

## Architecture

```
┌─────────────────────────────────────────────────────────────┐
│                    Four-Stage Pipeline                       │
│                                                             │
│  Stage 1 [Neural]    Stage 2 [Augment]   Stage 3 [Judge]   │
│  Phi-4-Mini          GCP ML Baseline     Llama 3.1 8B       │
│  i5 Worker Nodes  →  500k SynPUF     →  Ryzen Validators   │
│                                                    ↓        │
│                               Stage 4 [HITL] — This App    │
│                               Human Forensic Auditor        │
│                                    ↓                        │
│                          Gold Standard JSONL                │
│                          Bittensor Subnet Ready             │

```

## Stack

| Layer | Technology |
| :--- | :--- |

## Quick Start

```bash
# 1. Install dependencies
pip3 install -r backend/requirements.txt
pnpm install

# 2. Start everything
bash start.sh

# OR start separately:
# Terminal 1: Flask backend
cd backend && python3 app.py

# Terminal 2: React frontend
pnpm dev
```

Open http://localhost:3000 in your browser.

## API Reference

| Endpoint | Method | Description |
| :--- | :--- | :--- |
| `/api/health` | GET | Health check |
| `/api/traces/pending` | GET | Fetch pending traces for review |
| `/api/traces/stats` | GET | Dashboard statistics |
| `/api/traces/verify` | POST | Submit human decision (approve/deny/skip) |
| `/api/traces/generate` | POST | Generate new traces via pipeline |
| `/api/export/jsonl` | GET | Download Gold Standard JSONL |
| `/api/export/manifest` | GET | Export statistics manifest |

| `←` or `D` | Deny trace |
| `↑` or `S` | Skip trace |

## Connecting to Ollama (Production)

To replace the GCP ML baseline scenarios with live Ollama generation, update `backend/app.py`:

```python
import requests

def generate_via_ollama(prompt: str, model: str = "phi4-mini") -> str:
    res = requests.post("http://localhost:11434/api/generate", json={
        "model": model,
        "prompt": prompt,
        "stream": False,
    })
    return res.json()["response"]
```

Then call `generate_via_ollama()` in the `/api/traces/generate` endpoint.
